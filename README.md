# 一键搭建适用于Ubuntu/CentOS的IKEV2/L2TP的VPN

------
使用bash脚本一键搭建Ikev2的vpn

> * 服务端要求：Ubuntu或者CentOS-6.*

&nbsp;
> * 客户端：
 - iOS=>ikev1
 - Andriod=>ikev1
 - WindowsPhone=>ikev2
 - 其他Windows平台=>ikev2

&nbsp;
> * 可使用自己的私钥和根证书，也可自动生成

&nbsp;
> * 证书可绑定域名或ip

&nbsp;
> * 要是图方便可一路回车
